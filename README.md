# Compiler Project

Projeto de estudo e desenvolvimento de um compilador, com foco na implementação gradual de suas etapas fundamentais, como análise léxica, análise sintática, análise semântica e possíveis etapas posteriores de geração de código.

## Objetivo

Este repositório tem como objetivo documentar a construção de um compilador de forma incremental, servindo tanto como projeto acadêmico quanto como base de aprendizado sobre os principais componentes de um compilador.

## Etapas do projeto

- [x] Estrutura inicial do projeto
- [x] Primeira implementação simples do analisador léxico
- [ ] Expansão do conjunto de tokens
- [ ] Tratamento de erros léxicos
- [ ] Análise sintática
- [ ] Tabela de símbolos
- [ ] Análise semântica
- [ ] Geração de código ou interpretação
